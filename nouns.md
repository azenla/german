# Nouns

## German <=> English
| Noun | English |
| ---- | ------- |
| ich  | I       |
| du   | you |
| wir  | we |
| ihr | you (a group) |
| es | it |
| er | he |
| sie (singular)  | she |
| sie (plural) | they |
| Sie | you (formal) |

## Tips

- The noun 'du' always has a verb ending with -st. We call this rule the 'dust is eternal' rule.
